✈ 旅行手帳（trip-ledger）

多人旅行記帳工具：拍日文收據自動辨識金額並翻譯、手動快速記帳、 誰付的錢自動分帳、日圓即時換算台幣、預算進度與統計圖表。

打開旅行手帳 →
手帳剛才在休息，正在叫醒它（通常不到1分鐘）…

用的是免費雲端方案，15分鐘沒人用會小睡；第一次點開會自動叫醒，不用重新整理。

主要功能

  • 📷 拍收據 → 日文 OCR 辨識店名/金額 → 翻成繁中 → 確認後存檔
  • ✍️ 沒收據也能手動快速記一筆
  • 🤝 每筆可選「誰付的、誰分攤」，旅程結束一鍵算出誰給誰多少
  • 💱 日圓金額即時換算台幣參考價
  • 📊 預算進度、每日趨勢、分類占比、付款方式統計
  • 👥 多人共用，選頭像即可，不用註冊帳密

技術組合

FlaskNeon (Postgres)Render (Docker)Tesseract OCRChart.js
